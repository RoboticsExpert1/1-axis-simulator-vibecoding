# 🤖 Gemini x Python: 1-Axis Linkage Simulator

This project is a real-time robot kinematics simulator developed through a collaboration between a mechanical engineer and **Google Gemini**.

## 🛠 Features
* **AI-Assisted Coding:** Core logic and UI scaffolding generated via **Gemini**.
* **Expert Refinement:** Kinematic equations and parameters verified by an engineer with 8+ years of experience.
* **Interactive UI:** Real-time linkage control using `matplotlib` widgets.

## 🧠 Engineering Note
This repository demonstrates **"Vibe Coding"**—the process of using AI as a high-speed drafting tool while the engineer maintains total control over the logic, constraints, and final mechanical integrity.
